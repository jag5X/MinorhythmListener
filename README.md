minorhythm リスナー
======================
minorhythm リスナーは声優の茅原実里さんの Web ラジオ「[radio minorhythm](http://lantis-net.com/minorhythm/)」の聴取に特化したソフトウェアです。

動作環境
----------
本ソフトウェアの動作には .NET Framework 4.5 が必要です。
